# 🌍 Orbit & Clock (CSS Only)

Two mini projects in one: a CSS orbit animation and a CSS clock.

1. **Earth-Moon Orbit**  
   The Moon orbits around the Earth using only CSS keyframes and transforms.

2. **Clock**  
   A simple clock face with numbers.
